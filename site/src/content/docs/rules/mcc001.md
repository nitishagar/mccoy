---
title: MCC001 — Instruction injection marker
description: Catches instruction-injection markers in a tool description.
---

**Severity:** High

A tool description is shown to the model as context. If it contains instruction-like prose
(`ignore previous`, `system message`, `developer instruction`, `jailbreak`), a caller can use the
description itself as an injection vector.

## Fix

Remove instruction-like prose from the tool description. Descriptions should describe *what the
tool does*, not address the model.

## Note on the advisory pass

Some descriptions trip MCC001 but are benign documentation (e.g. "ignore previous examples in
tutorials"). With `OPENAI_API_KEY` set, the GPT-5.6 advisory pass annotates these as likely-benign
without changing the verdict.
