---
title: MCC002 — Open input schema
description: Catches input schemas that do not forbid unknown properties.
---

**Severity:** Medium

An input schema without `additionalProperties: false` accepts any field a caller sends, including
fields the tool never validated. Set `additionalProperties: false`.

## Fix

For Pydantic-backed FastMCP tools, set `model_config = ConfigDict(extra="forbid")` on the input
model. McCoy resolves nested `$ref`/`$defs` so a frozen inner model is recognized as closed.
